Using the Calculator

You can use the calculator to preform operators such as addition, subtraction, multiplication, division, modulus, square root, power, etc.

Type expressions into the text field located at the bottom of the window and press the 'Enter' button to submit
The inputted expression will be outputted and aligned left.
The answer will be outputted and will be aligned right. If the inputted expression is invalid, 'Invalid expression' will be outputted instead of an answer
  • This calculator follows the Order of Operations (PEMDAS)
  • A number divided by zero will either output 'Infinity' or '-Infinity'
  • Zero divided by Zero will output 'NaN' meaning 'Not a Number'
  • Both 'Infinity' and 'NaN' (case sensitive) are valid numbers that can be used in expressions

Variables

The following characters are the only valid characters for a variable name: abcdefghijklmnopqrstuvwxyz1234567890_
Variable names cannot have spaces in them.

Assignment

The format for assignment is:
{expression} -> {name}

{name} is a valid variable name {expression} can be a math expression or just a number. Examples:
5 + 5 * 5 -> x
(5 + 5) * 5 -> x
5 -> x

There does not have to be spaces around the '->'. Example:
5->x

'- >' also works (single space between - and >). Example:
5 - > x

Recall

Recall outputs the value of a variable, or "No variable for name" if a variable with the inputted name does not exist.

The format for recall is:
recall {name}

Deletion

Attempts to delete a variable, then outputs "Variable deleted" if deletion successful or "No variable for name" if a variable with the inputted name does not exist.

The format for deletion is:
delvar {name}

Menu Bar
